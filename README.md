ボイラープレート
====

[![Greenkeeper badge](https://badges.greenkeeper.io/sadnessOjisan/boilerplate-react-js.svg)](https://greenkeeper.io/)

ReactをJSで開発するためのボイラープレート

## 概要
アプリ開発用のボイラープレート。ライブラリ開発用ではない。

## 含まれるもの/採用しているもの
* gitignore
* prettier
* husky
* babel
* eslint
* airbnb-style-guide
* flow
* jest
* reselect
* redux
* redux-saga
* redux-logger
* axios
* flow
* webpack
* styled-components
* react-router

## 開発

```
# 起動
$ yarn run start:local

# テスト
$ yarn run test

# format
$ yarn run format

# lint
$ yarn run lint
```
